¿Como funciona el codigo?

El código esta conectado mediante la API de la aplicación Raider.io, la cual mide diferentes características de los jugadores del videojuego World Of Warcraft. En este programa se introduce el reino, nombre del jugador y servidor en el que juega para determinar algunas de esas cualidades de forma automática. A continuación, una lista de algunos jugadores y sus respectivas clases para poder probar el programa:
La estructura será: Nombre del jugador-Nombre del reino-Servidor
